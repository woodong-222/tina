# Tina - 티스토리 블로그 알림 디스코드 봇

티스토리 블로그 RSS를 모니터링하여 새 글을 자동으로 감지하고 디스코드에 알림을 보내주는 봇입니다. 주간 리포트, 벌금 관리, 마감 리마인드 등 블로그 모임 관리에 필요한 기능을 제공합니다.

## 실행 방법

### 방법 1. 도커 (추천)
도커가 설치된 환경에서 다음 명령어를 입력하여 실행합니다.

```bash
docker-compose up -d --build
```

### 방법 2. 로컬 실행
Python 3.12 이상의 환경이 필요합니다.

```bash
# 의존성 설치
pip install -r requirements.txt

# 봇 실행
python bot.py
```

## 환경 변수 설정 (.env)
실행 전 프로젝트 폴더에 `.env` 파일을 생성하고 다음 값을 채워넣어야 합니다.

- `DISCORD_TOKEN`: 디스코드 봇 토큰
- `BOT_MODE`: `test` (테스트모드) 또는 `prod` (운영모드)
- `PENALTY_AMOUNT`: 미작성 시 부과할 벌금 금액 (원)
- `RSS_POLL_INTERVAL`: 블로그 감지 간격 (분)

## 주요 명령어

### 일반 명령어
| 명령어 | 설명 |
|---|---|
| /신규등록 [블로그주소] | 본인의 티스토리 블로그를 봇에 등록합니다. |
| /삭제 | 등록된 본인의 블로그 정보를 삭제합니다. |
| /통계 [@유저] | 이번 주 및 이번 달의 포스팅 통계를 확인합니다. |
| /현황 [@유저] | 이번 주 작성 현황을 확인합니다. |
| /벌금 [@유저] | 현재까지 쌓인 벌금 현황을 조회합니다. |
| /새로고침 [@유저] | 블로그의 최신 글을 즉시 확인합니다. |
| /멤버목록 | 서버에 등록된 전체 멤버 목록을 조회합니다. |
| /도움말 | 봇의 사용법과 명령어 목록을 안내합니다. |

### 관리 명령어 (관리자 전용)
| 명령어 | 설명 |
|---|---|
| /멤버신규등록 @유저 [블로그주소] | 관리자가 특정 멤버를 대신 등록합니다. |
| /멤버삭제 @유저 | 관리자가 특정 멤버를 목록에서 삭제합니다. |
| /채널설정 #채널 | 알림이 전송될 디스코드 채널을 지정합니다. |
| /벌금설정 [금액] | 미작성 시 부과될 벌금 금액을 변경합니다. |

## 자동 알림 기능
- **블로그 감지**: 설정된 간격마다 새 글을 확인하여 자동으로 알림을 보냅니다.
- **주간 리포트**: 매주 월요일 09:00에 지난주 통계를 발송합니다.
- **마감 리마인드**: 매주 일요일 10:00에 글을 쓰지 않은 멤버에게 알림을 보냅니다.
- **환영 인사**: 새로운 멤버가 입장하면 봇 사용법을 안내합니다.
