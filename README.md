# Tina - 티스토리 블로그 알림 디스코드 봇

티스토리 블로그 RSS를 모니터링하여 새 글을 자동으로 감지하고 디스코드에 알림을 보내주는 봇입니다. 주간 리포트, 벌금 관리, 마감 리마인드 등 블로그 모임 관리에 필요한 기능을 제공합니다.

## 실행 방법
도커가 설치된 환경에서 다음 명령어를 입력하여 실행합니다.

```bash
docker compose up -d --build
```

## 환경 변수 설정 (.env)
실행 전 프로젝트 폴더에 `.env` 파일을 생성하고 다음 값을 채워넣어야 합니다.

- `DISCORD_TOKEN`: 디스코드 봇 토큰
- `DATABASE_URL`: PostgreSQL 연결 URL
- `BOT_MODE`: `test` (테스트모드) 또는 `prod` (운영모드)
- `GEMINI_API_KEY` (선택): AI 요약/태그/베스트픽용. 없으면 요약 기능 비활성화

> RSS 폴링 간격·요약 모델은 `config.py` 코드 고정값입니다.

## 주요 명령어

### 일반 명령어
| 명령어 | 설명 |
|---|---|
| /등록 | 본인 블로그 등록 (티스토리/벨로그 선택 후 주소 입력). |
| /삭제 | 등록된 본인의 블로그를 선택 삭제합니다. |
| /통계 [@유저] | 이번 주 및 이번 달의 포스팅 통계를 확인합니다. |
| /조회 [@유저] | 이번 주 작성 현황 및 포스팅 목록을 확인합니다. |
| /벌금 [@유저] | 현재까지 쌓인 벌금 현황을 조회합니다. |
| /새로고침 [@유저] | 블로그의 최신 글을 즉시 확인합니다. |
| /멤버목록 | 서버에 등록된 전체 멤버 목록을 조회합니다. |
| /랭킹 | 누적 작성 편수 명예의 전당(상위 3명)을 확인합니다. |
| /도움말 | 봇의 사용법과 명령어 목록을 안내합니다. |

### 관리 명령어 (관리자 전용)
| 명령어 | 설명 |
|---|---|
| /설정 | 알림 채널·초기화 요일/시간·벌금 금액·벌금 정지/재개를 한 곳에서 관리합니다. |
| /멤버등록 @유저 | 관리자가 특정 멤버를 대신 등록합니다 (플랫폼 선택). |
| /멤버삭제 @유저 | 관리자가 특정 멤버의 블로그를 삭제합니다. |
| /벌금변경 @유저 [금액] | 특정 멤버의 벌금을 수동 조정합니다 (양수: 추가, 음수: 차감). |
| /벌금정리 [정산\|초기화] | 미납 벌금을 정산(누적 유지)하거나 전체 초기화합니다. |

## 자동 알림 기능
- **블로그 감지**: 설정된 간격마다 새 글을 확인하여 자동으로 알림을 보냅니다.
- **주간 리포트**: 서버에서 설정된 초기화 시간에 지난주 통계를 발송합니다. (기본: 월요일 09:00)
- **마감 리마인드**: 초기화 시간 24시간 전에 글을 쓰지 않은 멤버에게 개인 DM으로 알림을 보냅니다. (기본: 일요일 09:00)
- **환영 인사**: 새로운 멤버가 입장하면 봇 사용법과 서버의 마감 시간을 안내합니다.
